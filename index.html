Munger Packages

APT repository and Homebrew tap for Munger software packages.

APT (Debian/Ubuntu)

curl -fsSL https://munger.github.io/packages/key.asc | sudo gpg --dearmor -o /usr/share/keyrings/munger.gpg
echo "deb [signed-by=/usr/share/keyrings/munger.gpg] https://munger.github.io/packages/ noble main" | sudo tee /etc/apt/sources.list.d/munger.list
sudo apt-get update
sudo apt-get install mirror-dedupe

Homebrew (macOS)

brew tap Munger/packages
brew install mirror-dedupe

Packages

  • mirror-dedupe — pool-based Debian/Ubuntu mirror syncing with hardlink deduplication

GPG Key
